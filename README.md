## Features

* Role-based access control (RBAC)
* Error and crash reporting
* CI/CD deployment pipeline
* Testing framework

## Getting Started

1. Install the dependencies:

```
npm install
```

2. Start the development server:

```
npm run dev
```

3. Build the application for production:

```
npm run build
```

4. Run the tests:

```
npm test
```
